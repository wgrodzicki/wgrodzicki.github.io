Cosmic: A Journey Among Shadows

This is a commercial project created and published by the indie game studio King's Pleasure. I formed part of the development team for over a year, with focus on C# scripting in Unity. My contribution includes implementation of some of the main character skills, improvement of the gamepad input, parts of the UI as well as parts of the translation and saving systems. I was also writing custom actions for visual scripting and worked with Steam API.

During the project I had an opportunity to use programming patterns such as: singleton pattern, observer pattern and object pooling. I was also doing my best to apply whenever I could the SOLID principles. Moreover, I worked with many Unity plug-ins, including Playmaker, DOTween, Odin Inspector or I2 Localization. I also got familiar with the Plastic SCM source control.

Watch the trailer on YouTube.
You can purchase the game on the official Steam page.

As Astra

This is my personal project with focus on 2D platforming, tilemaps and environmental challenges. The game consists of a short story and 3 unique levels of my own design. I developed and implemented gravity-related mechanic that strips the player of the core abilities and forces them to use the level itself to their advantage. I also included puzzle-like mechanics involving movable boxes and transporting platforms. The last part of the game features a final boss battle and special scripted events.

In the project I followed the OOP principles, made use of inheritance and polymorphism, as well as interfaces and object pooling.

Tech stack: Unity, C#, LINQ.

You can play the game online on Itch.io.
The whole project is available on my GitHub.

Math or Words?

This is my personal project developed in the relatively new .NET framework for multiplatform mobile and desktop apps - MAUI. The game has 2 modes: math-oriented and words-oriented. The first one tests player's proficiency in different math operations. The second one consists of word formation and working with associations. Player scores are stored in a local database and can be browsed on demand.

The project was an opportunity to learn about consuming APIs, in this case the Wikipedia API that serves as a validator of player answers. I used asynchronous programming when retrieving data from the Wikipedia and applied all CRUD operations to the database. I also took advantage of dependency injection for database connection.

Tech stack: .NET MAUI, C#, LINQ, Json.NET, XAML, SQLite, ADO.NET.

You can download the desktop build of the app here.
The whole project is available on my GitHub.

Space out of Control

This is a prototype created in Unity with a 5-person team during the 2023 TK Game Jam. The basic premise of the gameplay is to fight off enemies while watching out for periodical changes in the world space, which consist in switching the perspective from 3D/TPP to 2D/top-down. I was involved in the conceptualization and programming part of the project, focusing on the shooting mechanics.

You can download the game from Itch.io.

The Eradicator

This was my first full-stack web app. The front-end consists of the browser-based game, while the back-end allows to optionally create user account to store and browse scores.

The game was created from scratch in vanilla JavaScript without any additional frameworks, which thaught me a lot about fundamental concepts, such as manual sprite animation, creating hit boxes and parallax effects, using trigonometry for movement patterns and calculating the delta time. On the back-end I applied all CRUD operations to the database, used password hashing and session data storage. It was also a valuable experience to deploy the app to a real-life server.

Tech stack: JavaScript, HTML, CSS, Bootstrap, Python, Flask, Jinja.

Watch project showcase on YouTube.
You can play the game online here.
The whole project is available on my GitHub.

WinOXO

This project is a desktop app that can be seen as an expanded version of the tic-tac-toe game. I included customization options and a virtual opponent that enables a "solo mode" against the machine. Aside from the game itself I provided an option to create a user account to store scores, see current statistics regarding won, tied and lost games as well as the current place in the player ranking.

HabitHub

This project was my introduction to the ASP.NET Core framework. It allows the user to track and manage their habits. I also included a report feature which generates Excel spreadsheets with habit data.

During the development of the app I learnt about data models and blending C# with HTML through Razor syntax. I applied all CRUD operations to the database, made use of LINQ for ordering and JavaScript-specific features for datetime conversions.

The whole project is available on my GitHub.

Modal title

This project is a full-stack web app that simulates a warehouse management system. It can be used in the role of the warehouse coordinator or an employee, each with different access rights. The coordinator is responsible for managing wares and requests, while an employee can browse the items stored and place orders.

I applied all CRUD operations to the database and made extensive use of LINQ for ordering and sorting features. I also included many safety checks to validate user input before sending data to the database.

Tech stack: ASP.NET Core, C#, LINQ, Razor Pages, JavaScript, HTML, CSS, Bootstrap.

The whole project is available on my GitHub.

NumApp

This project is another classic - a calculator. Created in the multiplatform .NET framework for mobile and desktop apps, MAUI. To make it more interesting I challenged myself with the implementation of additional features: the user can apply the decimal precision of their choice, generate random numbers within the given range, convert numbers to binary and hexadecimal as well as save calculations locally in a JSON file.

During the development of the project I learnt about using streams to write to JSON files as well as XAML for user-friendly interfaces.

Tech stack: .NET MAUI, C#, Json.NET.

The whole project is available on my GitHub.

Alchemical Hustle

This is a gameplay mod I created for The Elder Scrolls III: Morrowind. I'd been always bothered by the fact that in the vanilla Morrowind the use of alchemical potions is very unbalanced and can be used to easily exploit the game. So I came up with a toxicity mechanic using MW script that restricts player's ability to drink potions by applying damage over time. The mod is compatible with the Morrowind engine replacer, OpenMW, and includes compatibility patches for other plug-ins.

The mod is quite popular, having reached over 9000 downloads.

You can download it from Nexus.

Dissident Ordinators

This is an immersion mod I created for The Elder Scrolls III: Morrowind during the 2023 May Modathon Month. I was inspired by a set of armor developed by another modder and decided to find a lore-friendly way of acquiring it during the game. In order to achieve that I modified some vanilla locations and designed a mini-quest that features the equipment as a reward. I also tweaked the 3D models using NifSkope. The mod is compatible with the Morrowind engine replacer, OpenMW.

The mod won the II place in the immersion category during the Modathon and was featured on Youtube.

You can download it from Nexus.

Morag Tong Armor Revisited

This is an equipment mod I created for The Elder Scrolls III: Morrowind. I was inspired by an older mod that introduced a new set armor into the game, but in many aspects was unfinished. I decided to expand on the initial idea, making it more immersive and polished. I also improved the models using NifSkope. The mod is compatible with the Morrowind engine replacer, OpenMW, and includes compatibility patches for other plug-ins.

The mod is pretty popular and has been translated by a fellow volunteer modder to Chinese.

You can download it from Nexus.

From the Flaming Realm

This is an quest mod I created for The Elder Scrolls III: Morrowind during the 2024 May Modathon Month. Its inspiration was an interesting set of armor developed by another modder. However, I felt that it needed a proper implementation into the game world and a backstory to make it lore-friendly. So I designed a questline revolving around the armor, wrote original dialogue lines for the NPCs, added scripted events using MW Script and improved the 3D models in NifSkope. The mod is compatible with the Morrowind engine replacer, OpenMW.

You can download it from Nexus.
